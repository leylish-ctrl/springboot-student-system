# Spring Boot Student System

## Project Description
This project is a simple REST API built with Spring Boot for managing student records.

## Technologies Used
- Java
- Spring Boot
- Maven
- REST API

## API Endpoints
GET /status → Check API status  
POST /students → Add student  
PUT /students/{id} → Update student  
DELETE /students/{id} → Delete student

## How to Run
1. Clone the project from GitHub
2. Open it in IntelliJ IDEA
3. Run DemoApplication.java
4. Open http://localhost:8080/status

Expected output: API is healthy

## Deployment
